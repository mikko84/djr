# Song Request Site

Welcome to the Song Request Site project! This application allows users to request songs in a simple and user-friendly interface.

## Features
- User-friendly interface
- Real-time song requests
- Easy navigation
- Admin panel for managing requests

## Getting Started
1. Clone the repository: `git clone https://github.com/mikko84/djr.git`
2. Navigate to the project directory: `cd djr`
3. Install dependencies: `npm install`
4. Start the application: `npm start`

### Built With
- [Node.js](https://nodejs.org/) - JavaScript runtime
- [Express](https://expressjs.com/) - Web framework for Node.js
- [MongoDB](https://www.mongodb.com/) - NoSQL database for storing song requests
